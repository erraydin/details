Keeper App

What is it?

Keeper is a clone of Google Keep web application. I created it using React. The source code can be found here

You can try it here. A google account is required to use the app.

Tools

1-React

2-Redux

3-Firebase (database and authentication)

4-Material ui components

5-Third party libraries, like this masonry component
